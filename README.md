## API Integration

The application integrates with the Gemini API using the `gemini-2.5-flash-lite` model. API key is configured in `src/services/aiService.ts`.

- Three.js (^0.180.0)
- @react-three/fiber (^9.3.0)
- @react-three/drei (^10.7.6)
